1. Включение флексов
  display: flex;

2. Свойства флексов

  2.1 Свойства контейнера
    flex-direction - меняет направление главной оси флекс контейнера
      row
      column
      row-reverse
      column-reverse

    justify-content - определяет положение элементов вдоль главной оси
      flex-start (default) - от начала главной оси до конца
      flex-end - от конца главной оси до начала
      center - выравнивает по центру оси
      space-between - свободное пространство порону разделяется между элементами
      space-around - свободное пространство размещается вокруг элемента с каждой стороны но х (между елемментами по факту будет 2х)
      space-evenly - свободное пространство порону разделяется между элементами ы станками контейнера

    align-items - определяет положение элементов вдоль поперечной оси
      stretch (default) - растягивает элементы вдоль поперечной оси
      flex-start
      flex-end
      center
      baseline

    flex-wrap - управляет переносом элементов 
      nowrap (default) -  перенос при переполнении не осуществляется
      wrap - переносит элементы на новую линию чтобы уместить все
      wrap-reverse - переворачивает направление побочной оси

    align-content - работает только если у вас включен врап, определяет свободное пространство между флекс-линиями, образованными при врапе
      flex-start
      flex-end
      center
      space-between
      space-around
      space-evenly
      stretch

    flex-flow - сокращение для flex-direction и flex-wrap

    gap - свойство, позволяющее указывать отступы между элементами
      1 значение (10px) - отодвигает элементы по горизонтали и вертикали
      2 значения (10px 20px) - отодвигает элементы по вертикали на 10 а по гроизонтали на 20

  2.2 Свойства флекс-детей ( > )
    order - целое число указывает порядок отображения элементов
      по умолчанию 0
    align-self - это аналог align-items но для конкретного ребенка
      stretch (default) - растягивает элементы вдоль поперечной оси
      flex-start
      flex-end
      center
      baseline
    flex-basis - управляет размером элемента вдоль главной оси

    content <= width/height <= max-width/height <= min-width/height

    width/height <= flex-basis <= content

    flex-grow
    flex-shrink
    flex
